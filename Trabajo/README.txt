Antes de compilar  (Probado en Lubuntu 24.04)

1. Asegurarse de tener g++ instalado
2. Instalar libopm-dev (Proporciona soporte para OpenMP)
3. Instalar CMake
4. Asegurarse de que esten los archivos "pd.csv" y "PEN_CLP.csv" 

Para compilar

1. Acceder al terminal, luego a la carpeta donde este el proyecto
3. Ejecutar "cmake ."
4. Una vez hecho cmake, se compila usando "make"
5. Por ultimo, ejecutar el programa utilizando "./mi_programa"
